OneDriveSDK

An example library for interacting with OneDrive. 
This library is implemented a Portable Class Library and can be used on a number of target platforms 
including .NET Framework 4.5, Windows 8, Windows Phone 8.1, and Windows Phone Silverlight 8.

This is just a copy, please get the current version from https://github.com/OneDrive/onedrive-explorer-win
